# space-invaders
A C++ Space Invaders clone based on a tutorial by @educ8s.

### Additonal features implemented by me:
1. Start "menu"
2. Pause button
3. Better "Game Over" screen
4. Game states
5. Separate UI class
6. Levels and infinite play
